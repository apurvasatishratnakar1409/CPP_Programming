# CPP_Programming
This repository contains C++ programs created for understanding the fundamentals of programming and Object-Oriented concepts.   The goal is to build strong logical thinking and problem-solving skills using C++.  
## Topics Covered
- Input / Output (cin, cout)
- Variables and Data Types
- Conditional Statements (if, else, switch)
- Loops (for, while, do-while)
- Functions
- Arrays and Strings
- Object-Oriented Programming (Classes & Objects)
- Constructors and Destructors
- Inheritance, Polymorphism, Encapsulation, Abstraction

## How to Compile and Run
1. Open terminal / command prompt in the directory where the .cpp file is stored.
2. Compile the program:
  g++ filename.cpp -o Myexe
3. Run the program:
  ./Myexe
4. on Windows:
   Myexe.exe
